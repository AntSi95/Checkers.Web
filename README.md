# ♟️ Checkers

Проект представляет собой онлайн-сервис для игры в шашки.

Взаимодействие компонентов реализовано по гибридной сетевой модели: real-time сессии игрового процесса подключены через SignalR, в то время как транзакционные, учетные и организационные задачи вынесены на легковесный HTTP/CRUD API слой.

Описание развития архитектуры, используемый глоссария, структуры слоев и технических контрактов находится в сопроводительном документе:

👉 **[Сопроводительный документ текущей разработки сервиса (docs/DEVELOP_DOC.md)](docs/DEVELOP_DOC.md)**
